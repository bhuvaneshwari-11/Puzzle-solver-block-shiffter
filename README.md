# 🧩 Puzzle Solver – Block Shifter

## 📌 Project Overview

Puzzle Solver – Block Shifter is an interactive web-based puzzle game where users rearrange shuffled image blocks to complete the original picture. The project demonstrates front-end development skills and logical problem-solving using JavaScript.

---

## 🚀 Features

* 🎮 Interactive image puzzle game
* 🔄 Smooth block/tile shifting
* ⏱️ Timer to track time spent
* 🎯 Move counter (moves remaining / used)
* 🏆 Scoreboard displayed after completing the puzzle
* 🎚️ **Multiple difficulty levels (Easy / Medium / Hard)**
* 👤 Player name input before starting
* 💾 Game state saved using Local Storage

---

## ▶️ How to Run

1. Download or clone this repository
2. Open `index.html` in any web browser
3. Enter your name and choose difficulty level
4. Start the game and solve the puzzle
5. View your score on completion

---

## 💡 How It Works

* The image is divided into blocks based on difficulty:

  * Easy → fewer blocks
  * Medium → moderate blocks
  * Hard → more blocks
* Blocks are shuffled randomly
* One empty space allows movement
* Click adjacent tiles to rearrange
* Complete the image to win

---

## 🏆 Scoreboard System

* Displays after puzzle completion
* Shows:

  * Player name
  * Time taken
  * Moves used / remaining
